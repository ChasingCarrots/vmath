Detailed Description

template<typename T, std::size_t M, std::size_t N = M>
class math::core::matrix< T, M, N >

A matrix with generic size and type. Column-major.

Template Parameters
TStorage type.
MNumber of rows.
NNumber of columns. Defaults to number of rows.
